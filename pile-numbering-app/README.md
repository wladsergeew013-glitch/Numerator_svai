# pile-numbering-app

MVP для инженерной нумерации свайного поля.

## Стек
- Backend: FastAPI + Pydantic
- Frontend: React + TypeScript + Vite + react-konva + Zustand
- Запуск: docker-compose

## Запуск
```bash
docker compose up --build
```

- Frontend: http://localhost:5173
- Backend: http://localhost:8000
